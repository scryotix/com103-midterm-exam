# com103-midterm-exam
This is a program that tracks the chores and their completion and says whoever is tasked with the chore.

To use this, type whoever's monitoring the room in the 'Room monitor name:' section, as well as your room number.

Then type out the number of the chore that aligns with the chore list. Then add the roommate's name, and it's completion status. It'll automatically produce a weekly chore report when you finish.
